Dice Tribes Log #8

12 February 2022

X-POST from itch devlog

After a few weeks of development, 0.3.0 is finally ready.

0.3.0 was initially planned to be the first paid version. However, after some considerations, I have decided to keep the web version free for the time being and the optional paid version for Windows and Mac. Linux will come later. With 0.3.0, the game is now renamed to Dice Tribes instead of Dice Kingdom to avoid confusion with a few other games. It also set a better expectation since we are not really building a kingdom.

Changes & Updates #

A couple of new buildings are added to the game.

  • Hunter’s Hut
  • Mushroom Farm
  • Altar

A new secret ambition is also added to the game along with new buildings etc.

With this version, all Tribes will also have unique Village building. This will give each Tribes a different play-style.

Roadmap #

With the paid version up, I would like to also list down the planned features up till 1.0.

  • 6 different tribes to choose from.
  • 10 different food buildings
  • 10 different utility buildings
  • 10 different ambitions (4 basic, 6 secret)
  • 6 different locations (currently 3 implemented)

The idea behind these is to provide different synergy between food buildings, utility buildings and ambitions; different food/utility combinations will make completing different ambitions easier.

Chaos Mode and Draft Mode #

Ultimately the game that I personally would like to play is Chaos mode and Draft mode.

In Chaos mode, the researches that is enabled will be randomly selected, together with a random starting village / configuration. This makes for a challenge as you need to decide which set of researches will work well together. In Draft mode, instead of it being random, you will be drafting the set of researches that you can use. The 6 basic tribes will essentially be more of a extended tutorial.

Alright, that’s all for this week.

Additional Thoughts:

After 3 weeks of work, 0.3.0 is finally up. Initially this version was meant to be the first paid beta and I was planning to keep the web version at 0.2.0. This is also why despite having 3 other build 0.2.1, 0.2.2, 0.2.3, none of it was released to the web.

However, I soon realised that not having weekly update makes the developments boring. Because of this, I decided that I will just keep the web version free for now with the latest update, along with paid Mac/PC version.

I also decided to rename the game from dice kingdom to dice tribes. There were a few reasons for this. The biggest reason is that there are a few games with that name that are relatively new, one in particular being an NFT game. I somewhat wants to avoid association :/ so I decided to just change the name.
